# 🤖 ChatBuddy: The AI ChatBot
<img width="1920" height="919" alt="Screenshot (206)" src="https://github.com/user-attachments/assets/ffb81edf-f65f-4c98-aa0e-c0626092d9c2" />

**ChatBuddy** is a simple AI-powered chatbot application designed to provide interactive conversations with users in real time. Built using the Gemini 2.5 Flash API, the chatbot can answer questions, assist users, and generate human-like responses through a clean and user-friendly interface.

## 🚀 Features

* 💬 **Real-Time AI Conversations** - Chat with an intelligent AI assistant instantly.

* 🧠 **Powered by Gemini 2.5 Flash API** - Generates fast and accurate AI responses using Google's Gemini model.

* ⚡ **Fast Response Generation** - Optimized for smooth and quick chatbot interactions.

* 🎨 **Simple & Clean Interface** - Minimal UI design for easy communication and better user experience.

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js
* **API:** Gemini 2.5 Flash API
* **Code Editor:** Visual Studio Code

## 📱 How It Works

1. User enters a message in the chat interface.
2. The app sends the query to the Gemini 2.5 Flash API.
3. The AI processes the request and generates a response.
4. The chatbot displays the AI-generated reply in real time.

## 🎯 Objective

The main objective of **ChatBuddy** is to create a simple and efficient AI chatbot that demonstrates the power of modern generative AI in interactive communication systems.
